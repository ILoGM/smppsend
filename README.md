SMPPSend
========

Simple utility for testing SMSC connections. It allows to bind to SMSCs, send submit_sm PDUs and wait for delivery reports.

Build
-----

    $ make

You need [Docker](https://www.docker.com/) installed to build.

Usage
-----

See

    $ smppsend --help
